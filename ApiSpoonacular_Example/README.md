# Spoonacular API Example

A small front-end project demonstrating how to fetch **random recipes** from the [Spoonacular API](https://spoonacular.com/food-api) and display them in the browser.

---

## Features

- Fetch random recipes with **API key authentication**.
- Display recipe **title and image** dynamically.
- Built with **HTML, CSS, and vanilla JavaScript**.
- Uses **Axios** for HTTP requests.

---

## How to Use

1. Open `index.html` in your browser.
2. Click the **"Get Random Recipe"** button.
3. A random recipe title and image will appear below the button.
4. Each click fetches a new random recipe.

> ⚠️ Note: If the API key becomes invalid, get a free key from [Spoonacular API](https://spoonacular.com/food-api) and replace it in `index.js`.
