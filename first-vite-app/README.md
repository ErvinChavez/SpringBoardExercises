# First Vite + React App

This project is a starter React application built using **Vite**.  
It demonstrates modern React development setup, component structure, ES module imports, and basic state management using hooks.

---

## Overview

This app was generated using Vite and configured with:

- React
- ESLint
- Vite development server
- Hot Module Replacement (HMR)

It includes a simple counter example and custom helper functions to demonstrate modular JavaScript imports.

---

## Features

- ⚛️ React functional component (`App.jsx`)
- 🔄 `useState` hook for state management
- 🔥 Hot Module Replacement (HMR)
- 📦 ES module exports/imports
- 🧹 ESLint configuration
- ⚡ Fast dev server powered by Vite
