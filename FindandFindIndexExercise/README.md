# Find & FindIndex Exercise

This exercise demonstrates the usage of the JavaScript array methods **`find`** and **`findIndex`** to locate elements in an array.

---

## Overview

- **`find`**: Returns the **first element** in an array that satisfies a given condition.
- **`findIndex`**: Returns the **index** of the first element that satisfies a given condition.

These methods are especially useful when searching arrays of objects.
