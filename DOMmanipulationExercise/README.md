# DOM Manipulation Exercise

A JavaScript exercise focused on practicing core **DOM manipulation techniques** using vanilla JavaScript.

---

## Overview

This project contains a series of 9 tasks that test different DOM manipulation skills, including:

- Changing text content
- Modifying inner HTML
- Updating styles
- Selecting elements by class
- Changing attributes
- Updating input values
- Using `classList`
- Creating and appending elements
- Removing elements from the DOM

Each task is automatically tested using a custom test script.

---

## What It Demonstrates

### DOM Methods & Properties Used

- `getElementById`
- `querySelectorAll`
- `innerText`
- `innerHTML`
- `style`
- `classList.add()`
- `appendChild()`
- `createElement()`
- `remove()`
- Attribute modification (`href`, `value`)

---

## How to Run

1. Open `index.html` in your browser.
2. The page will automatically execute:
   - `solution.js`
   - `test.js`
3. Each task will display:
   - ✅ if passed  
   - ❌ if failed  
4. Results are also logged in the browser console.

## Notes / Learning Takeaways

- Demonstrates understanding of direct DOM manipulation without frameworks.
- Shows difference between:
  - `innerText` vs `innerHTML`
  - Styling via JavaScript
  - Element creation vs string injection
- Uses both modern `.remove()` and traditional DOM removal patterns.
- Reinforces how JavaScript interacts directly with HTML elements.
