# API Food Generator

A simple web app that fetches and displays a random recipe from the [Spoonacular API](https://spoonacular.com/food-api) when you click a button.

---

## Features

- Click a button to generate a random recipe.
- Display the recipe title and summary directly on the page.
- Handles API errors gracefully with a clear error message.

---

## How It Works

- Built with **HTML**, **CSS**, and **vanilla JavaScript**.
- Uses **Axios** for making asynchronous API requests with `async/await`.
- Dynamically updates the DOM to show the recipe title and summary.
- Provides simple styling and error highlighting via CSS.

---

## How to Run

1. Clone or download this project.
2. Open `index.html` in your browser.
3. Click the **"Generate Recipe"** button.
4. A random recipe will appear in the main display area.

> ⚠️ Note: You need a valid Spoonacular API key in `index.js` (`spoonacularAPIKEY`) to fetch recipes.
